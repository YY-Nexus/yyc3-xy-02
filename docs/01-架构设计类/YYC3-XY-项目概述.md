## 项目简介

YYC3-XY（小语AI）是一个面向0-3岁婴幼儿的智能化成长守护系统，基于「五高五标五化」核心原则构建，融合医学、心理学、教育学多领域知识，为家长提供科学、个性化的育儿指导方案。

### 项目定位

- **目标用户**：0-3岁婴幼儿家长
- **核心价值**：科学育儿 + 智能陪伴 + 成长记录
- **服务模式**：AI智能助手 + 专业育儿指导 + 社区互动

### 项目愿景

成为家长最信赖的育儿智能助手，用科技守护每一个孩子的健康成长，让科学育儿触手可及。

## 核心目标

### 五高原则

| 高维度 | 实现方式 | 价值体现 |
|--------|---------|---------|
| **高前瞻性** | 预判发展阶段，提前规划成长路径 | 提前发现发展机会，及时干预 |
| **高整合性** | 融合医学/心理学/教育学多领域知识 | 提供全方位、科学化的育儿指导 |
| **高个性化** | 适配每个孩子的独特发展节奏 | 定制化育儿方案，精准匹配需求 |
| **高情感价值** | 关注亲子情感联结，记录温暖瞬间 | 增强亲子关系，留下珍贵回忆 |
| **高实操性** | 提供具体可执行的育儿指导方案 | 落地性强，家长易于执行 |

### 五标体系

| 标维度 | 实现方式 | 价值体现 |
|--------|---------|---------|
| **数据标准化** | 参考WHO等权威机构发展标准 | 确保数据准确性和权威性 |
| **发展标准化** | 基于发展心理学权威理论体系 | 科学评估孩子发展水平 |
| **安全标准化** | 遵循儿科安全规范与隐私保护 | 保障用户数据安全和隐私 |
| **记录标准化** | 采用统一格式的成长记录模板 | 便于数据分析和长期追踪 |
| **评估标准化** | 使用科学的评估工具与指标体系 | 客观评估孩子发展状况 |

### 五化架构

| 化维度 | 实现方式 | 价值体现 |
|--------|---------|---------|
| **阶段化** | 按0-22岁划分为多个发展阶段 | 针对不同阶段提供精准服务 |
| **模块化** | 将成长体系分解为可复用功能模块 | 灵活组合，满足多样化需求 |
| **场景化** | 针对具体育儿场景提供解决方案 | 贴近实际，解决具体问题 |
| **工具化** | 提供实用的记录工具与评估量表 | 便于家长日常使用 |
| **故事化** | 将成长记录转化为温暖的故事叙述 | 增强情感体验和记忆价值 |

## 功能特性

### 核心功能

#### 1. 智能成长记录
- **成长里程碑追踪**：记录孩子的关键成长时刻
- **发展评估**：基于科学标准评估孩子发展水平
- **数据可视化**：直观展示成长曲线和趋势
- **多维度记录**：身高、体重、语言、认知、社交等

#### 2. AI智能助手
- **智能对话**：24/7全天候AI咨询服务
- **个性化建议**：基于孩子情况提供定制化建议
- **育儿知识库**：权威育儿知识智能检索
- **情景问答**：针对具体育儿场景提供解决方案

#### 3. 科学育儿指导
- **发展阶段指导**：按月龄提供针对性指导
- **能力培养方案**：语言、认知、运动等能力培养
- **亲子互动建议**：增进亲子关系的互动活动
- **营养健康建议**：科学喂养和营养补充建议

#### 4. 社区互动
- **家长社区**：与其他家长交流经验
- **专家咨询**：在线咨询育儿专家
- **经验分享**：分享育儿心得和成长故事
- **话题讨论**：参与热门育儿话题讨论

#### 5. 语音交互
- **语音助手**：全天候语音交互服务
- **语音记录**：语音输入成长记录
- **语音查询**：语音查询育儿知识
- **语音提醒**：重要事项语音提醒

### 特色功能

#### RAG智能检索
- 基于向量数据库的语义搜索
- 精准匹配育儿知识
- 智能推荐相关内容
- 持续学习优化

#### 多模态交互
- 文字、语音、图片多模态输入
- 智能识别和理解
- 自然流畅的交互体验
- 个性化交互风格

#### 数据同步
- 多端数据实时同步
- 离线数据缓存
- 冲突自动解决
- 数据安全保障

## 快速开始

### 环境要求

- Node.js >= 20
- npm >= 9
- PostgreSQL >= 15
- Redis >= 7
- Docker >= 24

### 安装步骤

1. **克隆仓库**
```bash
git clone https://github.com/YY-Nexus/yyc3-xy-02.git
cd yyc3-xy-02
```

2. **安装依赖**
```bash
npm install
```

3. **配置环境变量**
```bash
cp .env.example .env
# 编辑.env文件，配置必要的环境变量
```

4. **初始化数据库**
```bash
npm run db:migrate
npm run db:seed
```

5. **启动开发服务器**
```bash
npm run dev
```

6. **访问应用**
```
前端: http://localhost:3000
后端: http://localhost:1229
```

### 开发命令

```bash
# 开发模式
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm run start

# 运行测试
npm run test

# 代码检查
npm run lint

# 类型检查
npm run typecheck

# 数据库迁移
npm run db:migrate

# 数据库回滚
npm run db:rollback
```
